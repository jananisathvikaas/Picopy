# 📋 Picopy

**Tagline: From Prompt to Paper.**

---

## 🧠 The Problem

AI chats are great for getting information — but terrible for studying.

Students can't easily bookmark a chat, highlight key points, print it, or read it offline on mobile. Long AI conversations get messy fast, buried in filler phrases like *"Sure, I can help with that!"* — making it hard to extract what actually matters.

---

## ✅ The Solution

**Picopy transforms your AI conversations into clean, organized, printable study documents.**

Paste any chat from ChatGPT, Claude, or Gemini — and Picopy instantly converts it into:

- 📄 Beautiful, structured **PDFs**
- 📝 Editable **Word documents**
- 📊 Ready-to-present **PowerPoint slides**
- 📚 Clean **study notes and flashcards**

No login. No payment. No API key. Works entirely in your browser.

---

## ⭐ What Makes Picopy Different

| Feature | What it does |
|---|---|
| 🏷️ Auto-titles & headings | Detects topics and creates chapters automatically |
| 🧹 Filler removal | Strips "Sure!", "Of course!" and keeps only real content |
| 📑 Table of contents | Added automatically for long documents |
| 🃏 Flashcard mode | Converts Q&A into flippable study cards |
| 📤 4 export formats | PDF, Word, PowerPoint, plain text |
| 📱 Mobile-friendly | Works on phone, tablet, desktop |
| 🔒 Private | Nothing is sent to any server — ever |

---

## 🚀 How It Works

```
You paste an AI chat
        ↓
Picopy detects questions, answers, and topics
        ↓
Formats into clean structured document
        ↓
You download as PDF / Word / PPT / Text
        ↓
Study anywhere, anytime — even offline
```

---

## 🛠️ Tech Stack

- Pure HTML, CSS, JavaScript — no framework needed
- jsPDF — PDF generation in the browser
- docx.js — Word file generation
- PptxGenJS — PowerPoint generation
- localStorage — daily usage tracking (no backend)
- Hosted free on Vercel
