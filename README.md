# 📋 Picopy — Your files & how to publish

## Files in this project:
- index.html — the main webpage
- style.css  — all the colors and design
- app.js     — all the working features

---

## How to publish FREE on Vercel (5 minutes):

1. Go to github.com and create a free account
2. Click "New repository" → name it "picopy" → click "Create"
3. Upload all 3 files (index.html, style.css, app.js)
4. Go to vercel.com → sign up free with your GitHub account
5. Click "Add New Project" → select your picopy repo
6. Click "Deploy" — done!
7. You get a free link like: picopy.vercel.app
